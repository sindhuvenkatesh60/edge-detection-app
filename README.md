# Edge Detection Android App

A powerful Android application for real-time edge detection in images using OpenCV and native C++ code.

## Features

- **Multiple Algorithms**: Canny, Sobel, and Laplacian edge detection
- **Real-time Processing**: Fast image processing using native C++ code
- **Camera Integration**: Capture images directly from camera
- **Gallery Support**: Process images from device gallery
- **Adjustable Parameters**: Customize edge detection thresholds
- **Save & Share**: Save processed images and share with other apps

## Technology Stack

- **Android Native** (Java/Kotlin)
- **OpenCV** for computer vision
- **C++/JNI** for high-performance image processing
- **CameraX** for camera operations
- **Material Design** for modern UI

## Build Instructions

1. Clone the repository
2. Open in Android Studio
3. Build the project (will download OpenCV automatically)
4. Run on device or emulator

## Permissions

- Camera: For capturing images
- Storage: For saving processed images
